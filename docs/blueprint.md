# **App Name**: CommerceVerse

## Core Features:

- Product Listing & Filtering: Display products with details, filter by category, and search functionality.
- Product Details Page: Show detailed product information, images, and add-to-cart option.
- Smart Product Recommendations: Suggest related products based on viewing history using an AI tool.
- Shopping Cart: Enable users to add, remove, and update quantities of products in a cart.
- Checkout Simulation: Implement a checkout process with form validation.
- Product Data Fetching: Fetching the data about products.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and security in e-commerce.
- Background color: Light blue (#E5EDFF), a desaturated tint of the primary color, creating a calming atmosphere for browsing.
- Accent color: Violet (#7830FF) to highlight key interactive elements such as buttons and links.
- Body font: 'PT Sans', a sans-serif font with a modern, approachable feel, ideal for readability in product descriptions and general content.
- Headline font: 'Space Grotesk', a contemporary sans-serif font perfect for drawing attention to product names and key headings.
- Use clean, modern icons to represent categories and actions, ensuring intuitive navigation.
- Design a grid-based layout for product listings, ensuring a clean and organized presentation.
- Subtle transitions and animations for adding items to the cart and updating the cart quantities to give positive feedback to user actions.
